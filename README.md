# Legendary-Emuhub-Android
The official Repository to Legendary Emuhub Androisd! Legendary Emuhub is a multi system emulator (WIP). It has many planned systems to come.

Steps to install
1. Go to releases and pick the version you wish to install
2. Download the APK and install it onto your device
3. When you launch the app, choose the core you want to use and select a ROM
4. After you choose your ROM, the game will boot along with on screen controls

NOTES:
Please keep in mind that this is a Work In Progress (WIP) and there may be bugs. More cores will be added in the future.
